# mask-type

Свойство **`mask-type`** определяет, используется ли элемент SVG `<mask>` в качестве яркости или альфа-маски. Это относится к самому элементу `<mask>`.

## Синтаксис

```css
/* Keyword values */
mask-type: luminance;
mask-type: alpha;

/* Global values */
mask-type: inherit;
mask-type: initial;
mask-type: revert;
mask-type: revert-layer;
mask-type: unset;
```
